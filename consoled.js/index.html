consoled.js

Fast, easy-to-use and independent module for styling your Nodejs console

Usage & SetupMethods & Features

Cross-platform

Consoled.js has been extensively tested on both Linux and Windows operating systems. This ensures that it is a reliable and versatile tool for developers working on different environments. With Consoled.js, developers can enjoy a consistent and stylish console output regardless of the operating system they are using.

No Dependencies

Unlike other similar tools, this module does not require any complex setup, and any other modules installed. All you need is Node.js installed on your system, and you can start using Consoled.js to create a visually appealing console for your server.

A wide range of methods

Consoled.js offers a variety of formatting options that developers can use to customize their console output. These include basic color text and background color text, as well as underlined, faded, and bright text. With options ranging from white, black, gray, red, yellow, green, cyan, blue, and magenta, developers can create console output that is both informative and visually appealing.
